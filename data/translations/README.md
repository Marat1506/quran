# Переводы на табасаранском языке

В этой папке хранятся JSON файлы с переводами сур Корана на табасаранский язык.

## Формат файлов

Файлы должны называться: `sura_{номер}_tabasaran.json`

Пример структуры файла:

```json
{
  "suraNumber": 70,
  "suraName": "Мягьражар",
  "ayahs": [
    {
      "ayahNumber": 0,
      "translation": "Бисмиллаhи-р-РахIмани-р-Рахим"
    },
    {
      "ayahNumber": 1,
      "translation": "Сала сагъа ахIлай..."
    }
  ]
}
```

## Примечания

- `ayahNumber: 0` используется для Бисмилляхи
- `ayahNumber` соответствует номеру аята в суре (1, 2, 3, ...)
- Если перевода для аята нет, он будет отображаться пустым
